# Template front-end starter pack

## Description

- Template front-end starter pack using Gulp modules

## Features

- Live reload with BrowserSync
- Compile Pug files to HTML
- Compile Stylus files to CSS
- Transpile ES6 JavaScript

## Setup

- Install modules
```
npm install
```

- Run Gulp
```
gulp
```
