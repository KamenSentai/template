# Template front-end starter pack

## Setup

### Require [Node.js](https://nodejs.org/en/) & [Gulp](https://gulpjs.com/)

- Get in `config` folder
```
cd [path/to/template/config]
```
- Install modules

MacOS
```
sudo npm install
```

Windows
```
npm install
```

- Run
```
gulp
```
